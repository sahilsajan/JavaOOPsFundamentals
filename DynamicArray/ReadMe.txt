Created Java code to create a dynamic array, which doubles in size after it reaches it array length size. The code has the following methods-

1)add(int element)- checks if array has reached size limit, if yes then doubles in size and adds next element, or simply adds next element
2)get(int index)- retrieves data at particular index
3)isEmpty()- checks if array is empty, then returns boolean
4)set(int index, int value)- inserts value in particular index
5)doubleCapacity()- doubles the capacity of the array
6)removeLast()- removes the last element of the array
7)size()- returns number of elements inserted in array
